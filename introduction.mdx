## Key Features

<AccordionGroup>
  <Accordion title="Secure Cross-Chain Transfers">
    - **EVM RPC canister** for reliable communication
    - **Threshold ECDSA signing** for secure validation
    - No off-chain dependencies required
    - Automatic refund processing for failed transactions
  </Accordion>
  
  <Accordion title="Comprehensive Token Support">
    - Lists over 1,000 EVM tokens across major chains
    - Real-time ICP token prices
    - Bridgeable tokens between ICP and EVM
    - Wrapped token tracking via LSM
  </Accordion>
  
  <Accordion title="Advanced Trading">
    - Multi-protocol DEX aggregation (Uniswap, PancakeSwap, SushiSwap)
    - Cross-chain routing with zero gas fees on ICP
    - Concentrated liquidity provision
    - Real-time price optimization
  </Accordion>
</AccordionGroup>
